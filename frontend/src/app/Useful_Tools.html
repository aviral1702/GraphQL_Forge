List of libraries/frameworks used during development: * For icons - [Font Awesome](https://fontawesome.com/) * For styling - [Bootstrap](https://getbootstrap.com/) * For routing - [React Router](https://reactrouter.com/) * For notifications - react hot toast * For notifications - notistack * For navbar - [mdb-react-ui-kit](https://mdbootstrap.com/docs/react/navigation/navbar/) * For navbar - material bootstrap * For forms - [Formik](https://formik.org/) * For validation - [Yup] * For react dom - [react-dom](https://reactjs.org/docs/react-dom.html) * For code editor - react-monaco-editor * For querying - [Apollo Client](https://www.apollographql.com/docs/react/) * For resource sharing - cors * For backend - [Node.js](https://nodejs.org/en/) * For backend - [Express](https://expressjs.com/) * For backend - [GraphQL](https://graphql.org/) * For backend - [MongoDB](https://www.mongodb.com/) * For backend - [Mongoose](https://mongoosejs.com/) * For running the server - [Nodemon](https://nodemon.io/)
